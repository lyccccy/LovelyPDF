# 🌙 夜间模式功能说明

## 功能介绍

这个 PDF 扩展现在只包含一个简洁实用的功能：**夜间模式**

在 PDF 查看器的右上角工具栏，你会看到一个月亮图标（SVG），点击即可切换日间/夜间模式。

---

## 使用方法

### 1. 重新加载扩展
```
1. 打开 chrome://extensions/
2. 找到 "My Custom PDF Viewer" 扩展
3. 点击刷新按钮 🔄
```

### 2. 打开 PDF 文件
- 可以是本地 PDF 文件
- 也可以是在线 PDF 链接

### 3. 切换夜间模式
- 在右上角工具栏找到月亮图标 🌙
- 点击切换到夜间模式（图标变为太阳 ☀️）
- 再次点击切换回日间模式（图标变为月亮 🌙）

---

## 图标说明

### 使用 SVG 图标
- **月亮图标**：清晰的矢量图形，支持任意缩放
- **太阳图标**：切换到夜间模式后显示
- **动画效果**：悬停时图标会放大，点击时会缩小
- **颜色自适应**：在夜间和日间模式下自动调整颜色

---

## 功能特点

✅ **一键切换** - 点击按钮即可切换模式  
✅ **自动保存** - 会记住你的选择，下次打开 PDF 自动应用  
✅ **护眼设计** - 深色背景减少眼睛疲劳  
✅ **智能反色** - PDF 内容颜色自动反转，保持可读性  
✅ **全局主题** - 工具栏、侧边栏统一深色风格  

---

## 视觉效果

### 日间模式 ☀️
- 白色背景
- 黑色文字
- 浅色工具栏

### 夜间模式 🌙
- 深色背景 (#1a1a1a)
- 浅色文字
- 深色工具栏
- PDF 内容智能反色

---

## 技术实现

- 使用 Chrome Storage API 保存偏好设置
- CSS 滤镜实现 PDF 内容反色
- 响应式设计，小屏幕自动隐藏按钮

## 常见问题

**Q: 找不到夜间模式按钮？**  
A: 确保已重新加载扩展，并且浏览器窗口宽度 > 768px

**Q: 夜间模式不生效？**  
A: 检查浏览器控制台是否有错误，尝试刷新页面

**Q: 设置会保存吗？**  
A: 是的，会自动保存，下次打开 PDF 时会应用你上次的选择
