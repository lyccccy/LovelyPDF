# 🔧 PDF 内容重叠问题修复

## 问题描述

在夜间模式下，PDF 内容出现两次，导致重叠，无法看清内容。

## 原因分析

之前的实现使用了 `invert` 滤镜来反转 PDF 颜色以适应夜间模式：
- 对 `.page` 应用了反色滤镜
- 对 `.textLayer` 也应用了反色滤镜
- PDF.js 的渲染机制导致内容层叠，造成重影

## ✅ 新的解决方案

**改用降低亮度的方式，而不是反色：**

### 旧方案（会重叠）
```css
filter: invert(0.9) hue-rotate(180deg);
```

### 新方案（不会重叠）
```css
filter: brightness(0.8) contrast(0.9);
```

## 🎨 新的夜间模式效果

### 深色背景
- 页面背景：`#2a2a2a`（深灰色）
- PDF 内容区：`#2a2a2a`

### PDF 内容调整
- **亮度**：降低到 80%（brightness(0.8)）
- **对比度**：降低到 90%（contrast(0.9)）
- **透明度**：90%（opacity: 0.9）

### 优点
- ✅ **不会重叠** - 不使用反色滤镜
- ✅ **内容清晰** - 保持原始内容的可读性
- ✅ **护眼效果** - 降低亮度，减少眼睛疲劳
- ✅ **性能更好** - 简单的滤镜比复杂的反色更高效

## 💡 技术细节

### CSS 滤镜效果

```css
/* 只对 canvas 应用滤镜 */
body.custom-night-mode #viewer .page canvas {
  opacity: 0.9 !important;
  filter: brightness(0.8) contrast(0.9) !important;
}
```

### 为什么这样不会重叠？

1. **只处理 canvas 元素** - PDF 的实际图像层
2. **不处理 textLayer** - 避免文本层的额外渲染
3. **不使用 invert** - 不改变颜色方向，只调整亮度

---

## 🎛️ 自定义调整

如果你觉得太亮或太暗，可以调整这些值：

### 更暗的效果
```css
filter: brightness(0.6) contrast(0.85);
opacity: 0.85;
```

### 更亮的效果
```css
filter: brightness(0.9) contrast(0.95);
opacity: 0.95;
```

### 保持原色（仅深色背景）
```css
/* 注释掉 filter 行 */
/* filter: brightness(0.8) contrast(0.9); */
opacity: 1;
```
