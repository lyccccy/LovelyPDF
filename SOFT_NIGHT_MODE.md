# 👁️ 夜间模式柔和配色方案

## 调整说明

为了减少眼睛疲劳，降低了夜间模式的对比度：

### 🎨 颜色调整对比

| 元素 | 旧配色（高对比） | 新配色（柔和） | 说明 |
|------|-----------------|---------------|------|
| **背景色** | | | |
| 主背景 | #1a1a1a (很黑) | #2d2d2d (中灰) | 调亮 |
| 工具栏背景 | #2a2a2a | #3a3a3a | 调亮 |
| 输入框背景 | #3a3a3a | #4a4a4a | 调亮 |
| **文字色** | | | |
| 主要文字 | #e0e0e0 (很亮) | #c0c0c0 (柔和) | 调暗 |
| 次要文字 | #a0a0a0 | #999999 | 调暗 |
| 禁用文字 | #666666 | #777777 | 调亮 |
| **边框色** | | | |
| 主要边框 | #4a4a4a | #5a5a5a | 调亮 |
| **其他** | | | |
| 悬停效果 | rgba(255,255,255,0.1) | rgba(255,255,255,0.08) | 降低 |
| 图标反色 | invert(1) | invert(0.85) | 降低 |

---

## ✨ 新配色特点

### 护眼优势
- ✅ **降低对比度** - 文字和背景的对比度从极高降至中等
- ✅ **柔和背景** - 从纯黑调整为中灰色，减少眼部刺激
- ✅ **柔和文字** - 从亮白调整为柔和灰白，减少眩光
- ✅ **柔和交互** - 悬停和点击效果更加柔和

### 视觉舒适
- 🌙 **长时间阅读** - 适合长时间阅读文档
- 👁️ **减少疲劳** - 显著降低眼睛疲劳
- 💤 **夜间友好** - 更适合夜间使用
- 🎯 **焦点清晰** - 内容仍然清晰可读

```
原配色：极致对比 → 视觉冲击强 → 短期清晰但长期疲劳

```

## 🔧 自定义调整

如果你觉得还需要调整，可以修改这些值：

### 更柔和（降低对比度）
```css
/* 背景再调亮一些 */
background-color: #404040 !important;

/* 文字再调暗一些 */
color: #b0b0b0 !important;
```

### 稍微增强（提高对比度）
```css
/* 背景调暗一些 */
background-color: #333333 !important;

/* 文字调亮一些 */
color: #d0d0d0 !important;
```

### 恢复原配色（高对比）
```css
/* 背景 */
background-color: #1a1a1a !important; /* 纯黑 */
background-color: #2a2a2a !important; /* 深黑 */

/* 文字 */
color: #e0e0e0 !important; /* 亮白 */
```

## 💡 使用建议

### 最佳实践
1. **环境光线** - 确保环境有适当的照明，不要在完全黑暗中使用
2. **屏幕亮度** - 调整屏幕亮度到舒适水平（约40-60%）
3. **阅读距离** - 保持50-70cm的阅读距离
4. **定时休息** - 每20分钟休息20秒，看20英尺外的物体

### 与其他功能搭配
- 🔆 **蓝光滤镜** - 配合系统的夜间模式或蓝光滤镜
- 📏 **字体大小** - 适当调大字体，减少眼睛负担
- 💻 **屏幕色温** - 调整屏幕色温到偏暖色调
